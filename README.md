Package Creation & Code Structuring / Pravljenje Paketa i Strukturiranje Koda

This project covers the basics of organizing Java code using packages. The focus was on understanding how to properly structure a codebase and follow standard Java conventions for package naming and code organization.

What I practiced:

Creating and using packages in Java

Understanding standard code structure conventions

Organizing classes across multiple packages
